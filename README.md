# Caesar-Cipher-Algorithm
Encrypt and decrypt text using the Caesar cipher with a configurable shift value (1-25)

**How to Run**
Install Flask if needed: pip install flask
Run: python app.py
Open browser to http://localhost:5000

**Example Usage**
With shift 5:
Encrypt: "Hello World" → "Mjqqt Btwqi"
Decrypt: "Mjqqt Btwqi" → "Hello World"

**With shift 13 (ROT13):**
Encrypt: "Hello World" → "Uryyb Jbeyq"
Decrypt: "Uryyb Jbeyq" → "Hello World"
